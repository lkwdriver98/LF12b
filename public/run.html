Install & Run

So startest du das Projekt auf frischen Serverknoten (Beispiel).

1) Bootstrap

OS vorbereiten, MicroK8s installieren, Add-ons aktivieren.

ssh user@cluster-master
sudo bash project/scripts/bootstrap.sh

2) Cluster bilden

Worker beitreten lassen, Status prüfen.

microk8s add-node   # master
microk8s join ...   # auf Worker
kubectl get nodes

3) Deploy

Nginx & Ingress ausrollen, Erreichbarkeit testen.

kubectl apply -f project/k8s/deploy-nginx.yaml
kubectl apply -f project/k8s/service.yaml
kubectl apply -f project/k8s/ingress.yaml
kubectl get pods -A
